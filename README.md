# 얼굴 성적기

얼굴 랜드마크 기반으로 `0.0 ~ 4.5` 범위의 재미용 성적 결과를 생성하는 엔터테인먼트 서비스다.

상세 스펙은 [Docs/mvp-spec.md](Docs/mvp-spec.md)를 따른다.

## 협업 기준

- 공식 명세와 API 계약 문서는 `Docs/`에 저장한다.
- Git 브랜치는 `main`, `front-end`, `back-end`만 사용한다.
- 프론트엔드와 백엔드는 각각 React, FastAPI 기준으로 분리해 작업한다.

## 원칙

- 본 서비스는 재미용이다.
- 실제 학업 능력, 성적, 성격을 판정하지 않는다.
- 결과는 얼굴 비율 특징, 룰 기반 계산, 랜덤 연출을 조합해 생성한다.
