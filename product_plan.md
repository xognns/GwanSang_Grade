# 얼굴 성적기

### 메인화면

- 이름 등록
- 얼굴 사진 등록
- 성적 조회 시작

### 얼굴 성적 조회 화면

- 사진 표시
- 성적 표시
- 코멘트
- 다시 조회 버튼
- 메인 화면으로 돌아가기

### 성적 측정

![image.png](attachment:b83c5935-bdd2-4c87-bce0-7edd949d8e8e:image.png)

이 사람을 0점으로 하고 0 ~ 4.5점 범위로 측정
아래는 바로 개발에 들어갈 수 있는 수준으로 정리한 관상 기반 “성적 측정(재미용)” 서비스 스펙 문서다. MVP 기준이지만 확장 고려까지 포함했다.

⸻

📄 1. 개요 (Overview)

서비스명 (가칭)

* FaceGrade / 관상성적측정기

목적

* 얼굴 이미지를 기반으로 재미용 학습 성향 및 성적 등급을 생성하는 엔터테인먼트 서비스

핵심 원칙

* 실제 능력/성적과 무관
* 결과는 확률적 + 연출 기반
* 유저 공유/바이럴 중심

⸻

🎯 2. 기능 요구사항 (Functional Requirements)

2.1 입력

* 사용자 얼굴 이미지
    * 업로드 (jpg/png)
    * 카메라 촬영 (선택)

⸻

2.2 처리 파이프라인

단계 1: 얼굴 검출

* 얼굴 영역 crop
* 단일 얼굴 기준 (MVP)

⸻

단계 2: 랜드마크 추출

사용 라이브러리:

* MediaPipe

출력:

* 468개 랜드마크 좌표 (x, y, z)

⸻
